Uso del Programa

Acceder a la Ventana Principal

Para acceder a la ventana principal, primero debes loguearte con un usuario y contraseña validos.

En caso de no tenerla, es posible crear una nueva.

Dentro de la ventana principal puedes usar el reproductor mp3 pulsando en el icono del altavoz.

Desde los menus puedes exportar la lista de animes. Tambien añadir, borrar o modificar los animes.

Desde la opcion Obtener Listado puedes buscar los animes pasando diferentes parametros al buscador.

Con el menu inicio, puedes volver al panel principal.


Uso del Reproductor

Para usar el reproductor, abrelo pinchando en el icono del altavoz correspondiente.

Selecciona una cancion y pulsa el boton de play.

Cierra la ventana y continua trabajando mientras escuchas la musica.

Tambien puedes hacer uso de los botones avanzar, retroceder, pausar y parar del reproductor segun necesites.


Agregar Animes

Para agregar animes hay que acceder al panel desde el menu correspondiente.

Se rellenan los campos con los datos requeridos.

Si no existen fecha de finalizacion u Ovas, se dejan los campos en blanco.

Con el boton insertar, se agrega el anime. Con el boton borrar campos se elimina el contenido de las cajas de texto.


Modificar Animes

Para modificar animes hay que acceder al panel desde el menu correspondiente.

Se introduce el nombre del anime a modificar y se pulsa el boton buscar.

Se rellenan los campos con los datos requeridos.

Con el boton modificar, se modifica el anime. Con el boton borrar campos se elimina el contenido de las cajas de texto.


Borrar Animes

Para borrar animes hay que acceder al panel desde el menu correspondiente.

Se introduce el nombre del anime a modificar y se pulsa el boton buscar.

Con el boton borrar, se borra el anime. Si no se pulsa buscar, el anime se puede borrar igualmente.


Obtener Listado Animes

Para buscar animes hay que acceder al panel desde el menu correspondiente.

Se selecciona el parametro de busqueda y se introduce en el campo de texto el parametro a buscar.

Pulsando el boton se obtiene el listado.


Exportar Datos

Con la opcion "Exportar Datos" la lista completa de animes se escribe en un archivo .txt o .rtf


Uso de la ayuda

Para acceder a la ayuda pulsar en la opcion correspondiente.

Para navegar a traves de ella usar los botones de control.

Uso de las notas

Pincha en anotaciones, selecciona la fecha y escribe la nota.

Las notas se guardaran con el nombre del usuario que las genero.

Para visualizar las notas pinchar en 'Ver Anotaciones'.
